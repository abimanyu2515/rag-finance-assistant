CHANGES MADE TO MAKE SIDEBAR AND HEADER FIXED WITH RESPONSIVE LAYOUT
=====================================================================

1. Created new file: contexts/SidebarContext.tsx
   - Created a React Context to share sidebar collapse state across components
   - Provides isCollapsed state and setIsCollapsed function
   - Used by Sidebar component and pages to sync layout changes

2. Modified: app/(root)/layout.tsx
   - Made it a client component ('use client')
   - Wrapped layout with SidebarProvider to enable context sharing
   - Removed hardcoded margin (ml-64) to make it dynamic

3. Modified: components/Sidebar.tsx
   - Changed from using local useState to useSidebar() hook from context
   - Changed positioning from 'relative' to 'fixed left-0 top-0'
   - Added z-40 for proper layering
   - Sidebar now stays fixed while scrolling

4. Modified: app/(root)/dashboard/page.tsx
   - Added useSidebar() hook to get collapse state
   - Made CustomHeader fixed with dynamic positioning:
     * left-20 when sidebar is collapsed
     * left-64 when sidebar is expanded
   - Made content area responsive with dynamic margin:
     * ml-20 when sidebar is collapsed
     * ml-64 when sidebar is expanded
   - Added transition-all duration-300 for smooth animations
   - Changed h-screen to min-h-screen to allow scrolling
   - Added mt-20 to account for fixed header

RESULT:
- Sidebar stays fixed on the left side while scrolling
- Header stays fixed at the top while scrolling
- Layout smoothly adjusts when sidebar collapses/expands
- All transitions are animated for better UX
